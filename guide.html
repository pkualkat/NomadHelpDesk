How To Use Putty

  1. Double-click on the putty.exe program or the desktop shortcut to launch the application.
  2. Enter your connection settings: ex(172.168.xx.xx)
    1. Port: 22 (leave as default)
    2. Connection Type: SSH (leave as default)
  3. Click Open to start the SSH session.
  4. If this is your first time connecting to the server from this computer, you will see the following output. Accept the connection by clicking Yes.
  5. Once the SSH Connection is open, you should see a terminal prompt asking for your username:
    1. login as
    2. Using keyboard-interactive authentication.
    3. Password:
  6. You are now logged into your server with SSH. You should see output like this: Input command: show logg
